Módulo 6 — Contábil

Relatórios automáticos, preenchimentos e GPTs especializados.

Como Realizar — DRE/Balanço com IA

  1. Exporte o balancete analítico por competência para uma planilha.
  2. Envie o arquivo à IA junto com o plano de contas e as regras de classificação da empresa.
  3. Solicite a geração da DRE/Balanço e um rascunho das notas explicativas padrão.
  4. Valide os saldos críticos e publique o relatório final em PDF.
Prompt — Geração de DRE 
Com base no balancete (anexo) e no plano de contas fornecido, gere a DRE do período [mm/aaaa].

Na sequência, anexe uma "Nota de Variações Significativas", com comentários curtos para cada conta com variação maior que 15% em relação ao período anterior.

Formato de saída: Tabela da DRE + Seção de comentários das notas.

Como Realizar — Preenchimentos Repetitivos

  • Crie modelos de texto (recibos, cartas, pareceres) com variáveis entre colchetes, como [NOME_CLIENTE].
  • Use a IA para preencher esses modelos em lote a partir de uma planilha (similar a um "mail merge").
  • Configure a automação para exportar cada documento preenchido em PDF.
Prompt — Mail merge contábil 
Use o template abaixo e a planilha de dados anexa para gerar um documento para cada linha da planilha.

Salve cada saída com o nome "[CNPJ]-[TipoDoc]-[Competencia].pdf" e, ao final, crie um índice com links para cada arquivo gerado.

Template:
[cole aqui o seu texto modelo com as variáveis, como por exemplo: [RAZAO_SOCIAL], [CNPJ], [VALOR], [COMPETENCIA]]
